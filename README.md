## 2020.4.11BIT第17届挑战杯

进度：

保存功能已解决

解决方法以及之前只能保存原始图片问题的原因:

​	**需要新建bitmap对象，然后使用最后的ColorMatrix矩阵给新建paint加上滤镜，然后将这个paint利用Canvas画给bitmap，这样才能获取到修改之后的bitmap。如果只是加滤镜，那么不会改bitmap本身**
